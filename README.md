# Sorted-Game

A cozy single-file mobile sorting game, built as a passion project for Android Chrome with a planned Capacitor app-store launch.

## About

Sorted is a relaxing game about sorting, collecting, and cozy progression. The full game lives in one self-contained HTML file — no build step, no dependencies, just open it in a browser.

Alongside the core game is a **trading-card expansion**: themed 50-card sets you collect by buying and opening packs, then organize in an in-game binder.

## Card Sets

| Set | Theme | Status |
|-----|-------|--------|
| Cozy Creatures | Gentle companion spirits | Complete (50/50) |
| Wild Kingdom | Real-world wildlife | Complete (50/50) |
| Realm Heroes | Fantasy | Complete (50/50) |
| Fossil Hunters | Dinosaurs | Complete (50/50) |
| Cosmic Frontier | Space — plants, aliens, ships, phenomena | In progress |

Each set follows the same rarity spread: 28 Common, 11 Rare, 7 Epic, 4 Legendary.

## Status

Active development. Core game and four card sets are complete and verified; the fifth set (Cosmic Frontier) is being built out now.

## Tech

Single HTML file. Card art is embedded directly in the file. No external assets or server required.
